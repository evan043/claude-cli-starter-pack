# /menu-for-happy-ui - Mobile Menu Alias

**This command is an alias for `/menu-happy`.**

## Instructions for Claude

When this command is invoked, **immediately invoke `/menu-happy`**.

This command exists for backward compatibility. The primary mobile menu command is now `/menu-happy`.

## Why This Alias Exists

- `/menu-for-happy-ui` was the original mobile menu command
- `/menu-happy` is the new, shorter name
- Both commands show identical mobile-formatted menus
- Use whichever you prefer

## Related Commands

- `/menu-happy` - Primary mobile menu (preferred)
- `/menu` - Desktop menu (always desktop format)
- `/happy-start` - Initialize Happy Mode session
