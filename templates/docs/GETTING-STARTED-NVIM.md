# CCASP Neovim Plugin - Getting Started

## Quick Start

1. Install the plugin (lazy.nvim):
   ```lua
   { dir = "~/.local/share/ccasp/nvim-ccasp", config = true }
   ```

2. Run `:CcaspWelcome` to open the interactive Getting Started guide

3. Navigate with `[n]ext`, `[p]rev`, or number keys `[1-8]`

## Key Features

| Feature | Access |
|---------|--------|
| Getting Started | `:CcaspWelcome` |
| Help Panel | `:CcaspHelp` or `<leader>c?` |
| Sidebar | `Ctrl+B` |
| Dashboard | `<leader>cd` |
| Session Manager | `Ctrl+Shift+N` |

## Help System

Access help anytime with `:CcaspHelp` - includes 11 searchable wiki topics covering all features.

## Health Check

Run `:CcaspHealth` to verify your installation.
