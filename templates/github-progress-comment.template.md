# GitHub Progress Comment Template

This template defines the format for progress update comments posted to GitHub issues by the orchestrator.

## Variables

- `{{plan_name}}` - Name of the phase development plan
- `{{completion_percentage}}` - Overall completion percentage (0-100)
- `{{phases}}` - Array of phase objects
- `{{current_phase}}` - Currently active phase
- `{{completed_item}}` - What was just completed (task or phase)
- `{{timestamp}}` - ISO timestamp of update

## Comment Template

```markdown
## Phase Development Progress Update

```
[{{progress_bar}}] {{completion_percentage}}%
```

{{#if completed_item.type === 'task'}}
**Task Completed:** `{{completed_item.taskId}}`
{{#if completed_item.summary}}
> {{completed_item.summary}}
{{/if}}
{{/if}}

{{#if completed_item.type === 'phase'}}
**Phase Completed:** {{completed_item.phaseName}} (`{{completed_item.phaseId}}`)
{{/if}}

### Phase Status

| Phase | Status | Progress |
|-------|--------|----------|
{{#each phases}}
| {{status_emoji}} {{name}} | {{status}} | {{tasks_done}}/{{tasks_total}} |
{{/each}}

{{#if current_phase}}
### Current Phase: {{current_phase.name}}

{{#each current_phase.tasks}}
- [{{checkbox}}] `{{id}}`: {{title}}
{{/each}}
{{/if}}

---
*Automatically synced by CCASP Orchestrator*
*{{timestamp}}*
```

## Phase Completion Template

```markdown
## Phase Completed!

**{{phase_name}}** (`{{phase_id}}`) has been completed.

### Summary
- Tasks completed: {{tasks_completed}}/{{tasks_total}}
- Duration: {{duration}}
- Agent: {{completed_by}}

{{#if success_criteria}}
### Success Criteria Met
{{#each success_criteria}}
- [x] {{this}}
{{/each}}
{{/if}}

### Next Phase
{{#if next_phase}}
Starting: **{{next_phase.name}}** ({{next_phase.tasks.length}} tasks)
{{else}}
No more phases - plan nearing completion!
{{/if}}

---
*Overall progress: {{completion_percentage}}%*
```

```markdown
### Error
```

{{error_message}}
```

{{#if stack_trace}}

<summary>Stack Trace</summary>

```

```
</details>

### What Was Attempted
{{attempted_action}}

### Recommended Action
{{#if retryable}}

{{/if}}
- [ ] Investigate the error
- [ ] Mark as blocked if external dependency
- [ ] Skip if non-critical

---
*Phase progress paused pending resolution.*
*Posted at {{timestamp}}*
```
