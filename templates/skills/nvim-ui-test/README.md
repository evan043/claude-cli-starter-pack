# Neovim UI Test Skill

This skill provides knowledge and procedures for testing the nvim-ccasp Neovim plugin UI.

## Skill Contents

- `nvim-ui-test-skill.md` - Complete testing guide with procedures and troubleshooting

## When to Use

- Running UI tests for the nvim-ccasp plugin
- Debugging UI test failures
- Writing new UI tests
- Understanding the test harness architecture

## Quick Start

```
/ui-test          # Run full test suite
/ui-smoke         # Quick smoke test
/ui-snapshot      # Capture layout state
/ui-bug           # Report a bug
/ui-fix           # Fix a failing test
```
